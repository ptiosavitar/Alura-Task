# Aprendendo JavaScript - Pedro

Olá, meu nome é Pedro e este repositório contém algumas tarefas que fiz enquanto aprendo JavaScript.

## Tarefas

1. **Verificador de Dia da Semana**  
   Um script que verifica se hoje é fim de semana e exibe uma mensagem apropriada.

2. **Verificador de Números Positivos ou Negativos**  
   Um script que identifica se um número digitado pelo usuário é positivo ou negativo.

3. **Jogo dos Pontos**  
   Um pequeno jogo que verifica se a pontuação do usuário é suficiente para ganhar.

4. **Banco Simulado**  
   Simula uma retirada de dinheiro de uma conta bancária fictícia.

5. **Boas-vindas ao Usuário**  
   Pede o nome do usuário e exibe uma mensagem de boas-vindas.
